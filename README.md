# Setup

Download and install anaconda from https://www.anaconda.com/products/individual

```
conda create -n rasa-server python=3.8
```

```
conda activate rasa-server
```

```
pip3 install -U pip
```

```
pip install rasa
```

# Runing API server

```
rasa run
```

# Running on command line

```
rasa shell
```
